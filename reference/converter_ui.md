# Dataset Converter Module – UI

Renders a two-column card layout: a conversion form on the left and a
format-reference table plus output preview on the right.

## Usage

``` r
converter_ui(id)
```

## Arguments

- id:

  Character string. The Shiny module namespace identifier.

## Value

A [`shiny::tagList()`](https://rdrr.io/pkg/shiny/man/reexports.html)
with the converter UI.
