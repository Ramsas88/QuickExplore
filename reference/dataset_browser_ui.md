# Dataset Browser Module – UI

Renders the sidebar panel that lets users add/remove directory-based
libraries and select a dataset to load.

## Usage

``` r
dataset_browser_ui(id)
```

## Arguments

- id:

  Character string. The Shiny module namespace identifier.

## Value

A [`shiny::tagList()`](https://rdrr.io/pkg/shiny/man/reexports.html)
containing the sidebar UI elements.
